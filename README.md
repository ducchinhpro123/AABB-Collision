# AABBCollision

A C++ project for AABB (Axis-Aligned Bounding Box) collision detection using SFML.

![img](2025-04-07_10-10.png "Demo")

## Building

```bash
# Configure the build
meson setup builddir

# Compile the project
meson compile -C builddir

# Run the executable
./builddir/aabb_collision
```
